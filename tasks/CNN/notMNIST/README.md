# Классификация изображений с использованием CNN для датасета notMNIST

## Описание
Этот проект демонстрирует использование свёрточной нейронной сети (CNN) для классификации датасета notMNIST. В проекте выполняются загрузка данных, их предобработка, обучение модели, оценка и визуализация результатов.

Свёрточные нейронные сети (CNN) — это вид нейронных сетей, специально разработанный для анализа данных с сетчатой структурой, например изображений. Они используют свёртки для выделения признаков и пулинг для уменьшения размерности данных. Благодаря этим механизмам CNN эффективно применяются в задачах классификации, распознавания объектов и обработке изображений.

## Детали задачи
1. **Подготовка данных:**
   - Загрузка датасета notMNIST из сериализованного файла PyTorch.
   - Разделение датасета на обучающий (80%) и валидационный (20%) наборы.
2. **Определение модели:**
   - CNN с двумя сверточными слоями, max-pooling, активацией ReLU, dropout и полносвязными слоями.
3. **Процесс обучения:**
   - Модель обучается в течение 10 эпох с использованием оптимизатора Adam и функции потерь cross-entropy.
4. **Метрики оценки:**
   - Отслеживаются потери на обучении и валидации.
   - Рассчитывается точность на валидационном наборе после каждой эпохи.
5. **Визуализация:**
   - Генерируются графики потерь, динамика точности, матрица ошибок и примеры предсказаний.
